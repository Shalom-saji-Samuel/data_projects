# Sakila dashboard

### Description 
a dashboard on the sakila dataset , done on power bi 

## Objectives

- Use this project as practice
- Add this to my portfolio

---

## Tools used

- Power bi
- Sakila Sample Database
- Git & GitHub

---
## Skills shown 
- Connecting mysql database to power bi 
- Dashboard making 
- KPI cards 
- Slicers 
- Tooltips to show additional information

---
## Key insights 
Some insights we can get from this dashboard are -
- Total revenue is $37K
- The store has 326 customers 
- The rentals are highest during July and August 
And many more information
